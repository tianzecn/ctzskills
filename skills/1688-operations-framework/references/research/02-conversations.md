# 02. 采购买家场景、询盘/转化/客户运营、B类买家路径

调研角色：1688 运营主题 Skill / Agent 2
调研日期：2026-05-11
范围：采购买家意图、B 类买家路径、询盘与转化承接、客户运营工作流
排除来源：知乎、微信公众号、百度百科

## 结论摘要

1688 买家侧正在从“搜索货源并下单”的单点动作，变成“找货/挑品/询单/下单/履约/复购/换供”的连续采购工作台。对商家运营来说，核心不是只提高曝光，而是把买家的采购意图结构化承接：识别买家类型、还原采购阶段、补齐信任与履约证据、缩短询盘到报价的响应时间，并把复购、补货、换供、SKU 变化监控沉淀成客户运营流程。

## 买家画像与核心意图

### 1. 传统电商/内容电商分销买家

典型来源包括淘宝、抖音、快手、小红书、微信等下游电商或内容平台开店的分销商。开放平台把这类买家的链路定义为：1688 优质代发货源一键多店铺货、下游订单自动回流采购、自动发货。其本质意图不是“买一批货”，而是降低选品、铺货、下单、发货的人工成本。

可运营化发现：

- 商品页必须明确是否支持一件代发、包邮、发货时效、退换政策、库存稳定性。
- 客服话术要先判断买家是否为下游店铺经营者，再追问店铺平台、日单量、是否需要图片包/数据包/铺货授权。
- 对高潜分销买家，应从单次询单转为“货盘合作”：提供爆品清单、上新节奏、价格梯度、库存预警和售后边界。

### 2. 跨境电商采购商

开放平台面向跨境电商平台开店的 1688 采购商，强调智能选款、一键铺货、自动下单、批量支付和订单信息同步。AIBUY 插件则直接暴露了全球买家的痛点：多语言商品理解、汇率换算、跨境属性字段、旺旺对话翻译。跨境宝页面进一步说明，海外公司主体买家需要把 1688 ID、离岸人民币支付、担保交易和 ERP 批量支付接起来。

可运营化发现：

- 跨境买家更在意“能否卖到海外”而不只是国内批发价，详情页应补充重量、尺寸、包裹属性、材质、认证、是否可贴牌/换包装等字段。
- 询盘表单应结构化收集目标市场、平台、预计采购量、是否需要中性包装、是否需要质检/合规文件。
- 对海外主体买家，报价单要区分货价、国内物流、打包要求、交期、可提供文件，不要只发一个单价。

### 3. 小 B/轻创业/社群与内容带货买家

亿邦动力披露，近年新增买家包括小红书博主、抖音主播、快手达人、社区团购团长、微商、群主、小店店主、B站 UP 主、宝妈、大学生、新锐白领等；其高频需求包括源头好货、品牌平替、工厂定制，并呈现高频小额采购特征。

可运营化发现：

- 这类买家常处在“试卖/测款”阶段，首次成交门槛比利润率更关键。
- 运营应准备小起订量、样品、混批、快返、定制起订量、可复购周期等信息。
- 对内容型买家，素材交付能力会影响转化：主图、短视频素材、卖点文案、达人可讲解点、包装展示都应作为询盘承接内容。

### 4. 企业/工业品采购买家

艾瑞报告称 1688 已汇聚 6000 多万 B 类买家，围绕“用户为先、AI驱动、做强供应链”提升供需匹配与交易效率，并强调智能采购助手、智能需求发布、智能图片搜索、智能选货、数字化产地地图、VR 在线探厂等能力。企业采购/工业品买家的意图通常更接近“确定供应商”：要验证资质、产能、交期、稳定供给、售后和账期。

可运营化发现：

- 工业品和企业采购详情页要标准化参数、型号、适配场景、资质证书、检测报告、售后条款。
- 客服应快速区分“询价比价型”“技术确认型”“项目采购型”“长期供应型”买家，并给出不同跟进节奏。
- 对项目型买家，转化关键不是立即下单，而是形成 RFQ/报价/样品/合同/交期确认的闭环。

## B 类买家采购链路

### 阶段 1：需求触发

常见触发包括下游订单增长、爆品跟进、内容平台热品、老货断供、价格上涨、供应商履约不稳定、企业项目采购、跨境平台选品。AI 采购助手、智能选货、图搜找同款和产地地图会让买家更快进入候选供应商池。

运营工作流：

- 建立“买家来意标签”：补货、换供、测款、代发、定制、项目采购、跨境采购。
- 对每个标签配置首轮问题，不用同一套欢迎语承接所有买家。
- 在商品标题、属性、主图和详情首屏中覆盖买家触发词：现货、代发、源头工厂、可定制、48小时发货、包邮、退换、资质、跨境。

### 阶段 2：找货与筛选

买家会通过搜索、图搜、平台推荐、货盘、AI 导购、产业带/产地线索筛选供应商。开放平台提到货源推荐提供一件代发包邮、48小时内发货、7天包退等服务保障货源池；这说明履约承诺已成为买家筛选的一部分。

运营工作流：

- 商品信息标准化：标题、类目、属性、SKU、价格梯度、库存、起订量、发货地、发货时效保持一致。
- 详情页放“筛选证据”：工厂/仓库/产线、质检、认证、真实发货时效、售后政策、客户案例。
- 对支持代发/跨境/定制的商品，建立单独货盘，避免买家反复询问基础条件。

### 阶段 3：询盘与咨询

B 类买家询盘通常围绕 MOQ、价格梯度、交期、库存、定制、样品、发票、账期、售后、认证、包装、物流、是否可长期合作。跨境买家还会追问语言、汇率、包裹属性、目标市场合规、ERP/批量支付与订单同步。

运营工作流：

- 首响模板必须先完成分流：采购用途、平台/渠道、预计数量、是否定制、交期、收货地/出口地。
- 报价模板要结构化：产品规格、阶梯价、MOQ、样品价、打样周期、生产周期、库存有效期、物流方式、售后边界。
- 建立“询盘质量评分”：明确 SKU/数量/交期/用途/收货地的买家优先跟进；仅问“多少钱”的买家用轻量模板引导补充信息。

### 阶段 4：报价、样品与信任建立

企业采购和跨境买家会把供应商稳定性纳入决策。艾瑞报告提到数字化产地地图、VR 在线探厂、商机图谱、工厂图谱等基础设施，说明平台正在把“源头可信度”产品化。

运营工作流：

- 对高意向买家提供“信任包”：营业执照、工厂/仓库图、质检报告、合作案例、发货记录、售后规则。
- 样品流程要明确：样品是否收费、是否可退、打样周期、样品确认后大货价是否抵扣。
- 对大额/长期买家，跟进动作应从旺旺聊天升级到报价单、样品单、合同条款、交期排产表。

### 阶段 5：下单、支付与履约

跨境宝显示，境外主体买家需要注册万里汇外贸 B2B 账号、绑定 1688 ID、兑换离岸人民币，并在 1688 下单时选择“担保交易--跨境宝”付款；同时支持 API 对接，实现 ERP 后台批量支付。亿邦动力报道也提到，1688 对送达时间、按时发货、自动理赔、退货包运费等履约体验做了治理。

运营工作流：

- 下单前确认：SKU、数量、地址、发票、发货时效、是否拆单、是否需要代发面单。
- 发货中同步：揽收时间、物流单号、异常预警、缺货替代方案。
- 售后承接：退换边界、运费责任、破损补发、少件漏发处理时限。

### 阶段 6：复购、补货与客户运营

亿邦动力提到 1688 的智能货源管理覆盖复购、换供、组货、价格与 SKU 变化监控。对商家来说，客户运营要从“成交后结束”变成“成交后进入货盘关系”。

运营工作流：

- 成交后给买家打标签：试样、测款、稳定复购、代发客户、跨境客户、定制客户、项目客户。
- 建立补货提醒：按历史采购周期、库存消耗、下游活动节点、节日季节性推送。
- 建立换供预警：当价格、SKU、库存、发货时效变化时主动通知重点买家。
- 建立复购内容包：新品、热卖、平替、同类组合、利润测算、定制案例。

## 可转化为 Skill 的运营工作流模块

### 模块 A：买家意图识别器

输入：买家首句、商品链接、询问字段、历史订单。
输出：买家类型、采购阶段、意图标签、优先级、下一句追问。

规则示例：

- “能不能代发/有没有图片包/能铺货吗” → 分销代发买家，追问平台、店铺数量、日单量、面单要求。
- “有没有英文资料/重量尺寸/能不能发海外” → 跨境买家，追问目标国家、平台、包装、认证。
- “能不能定制 logo/小批量试做” → 定制/小 B 测款，追问数量、工艺、交期、素材文件。
- “参数表/检测报告/发票/账期” → 企业采购，追问应用场景、项目周期、采购流程。

### 模块 B：询盘承接 SOP

1. 30 秒内识别买家来意与采购阶段。
2. 用 3-5 个问题补齐报价必需信息。
3. 输出结构化报价，不只报单价。
4. 补充履约与信任证据。
5. 设置下一步动作：样品、下单、打样、发合同、约时间确认。

### 模块 C：B 类买家报价模板

字段：

- 买家类型：分销/跨境/小 B/企业采购/定制。
- 产品与规格：SKU、材质、尺寸、颜色、包装。
- 价格：阶梯价、样品价、定制费、运费口径。
- 交付：现货数量、生产周期、发货时效、物流方式。
- 保障：退换规则、质检、发票、售后时限。
- 下一步：样品确认、打样资料、下单链接、合同/对账流程。

### 模块 D：成交后客户运营

触发器：

- 成交后第 1 天：确认收货预期和售后入口。
- 成交后第 7 天：确认试卖/使用反馈。
- 成交后第 14/30 天：按品类周期提醒补货。
- SKU/价格/库存/发货时效变化：自动通知重点买家。
- 节日/平台活动前：推送备货清单和组合货盘。

## 对 1688 运营 Skill 的提示词建议

在处理买家对话时，不要只生成客服话术。应先判断买家属于哪类 B 类路径，再决定承接策略：

1. 买家是来比价、找货、测款、代发、定制、跨境采购，还是企业项目采购？
2. 当前处于需求触发、找货筛选、询盘、报价样品、下单履约、复购补货哪一阶段？
3. 缺哪些信息会阻塞报价或成交？
4. 哪些证据能降低买家风险感：价格、时效、资质、案例、售后、支付、物流？
5. 下一步动作是什么：追问、报价、发样、建货盘、下单、售后、复购提醒？

## 风险与待补证据

- 1688 官方页面大量内容依赖登录或动态渲染，部分细分规则页无法稳定抓取；当前研究优先使用开放平台首页、AIBUY 插件页、跨境宝页面及高可信行业报道。
- 亿邦动力文章包含平台披露与媒体解读混合内容，适合作为趋势与买家画像证据，但具体运营指标应在后续用 1688 商家后台或官方规则二次校验。
- 不同类目的买家链路差异很大，后续 Agent 可按服饰、工业品、家居百货、跨境小商品分别补充分品类询盘模板。
